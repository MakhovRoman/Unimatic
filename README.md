# Unimatic

Проект по созданию приложения для заметок. Включает в себя серверную и клиентскую части.

Реализован следующий функционал:
- авторизация
- регистрация
- создание / изменение / удаление заметок

## Scripts

Запуск сервера
```
npm run dev:server
```

Запуск клиента
```
npm run dev:client
```
